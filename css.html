Selectors:

1. * - Universal Selector:

The universal selector is used to select all the elements of the entire webpage.

Syntax:

* {
your code...
}

2. , - Type or Element Selector:

Type or Element is used to select one or more than one elements by using their tag name. Tags are separated by commas.

Syntax:

h1, h2 {
your code...
}

3. . - Class Selector:

Class Selector is used to select elements by their Class Names.

Syntax:

.abc {
your code...
}

4. # - ID Selector:

ID Selector is used to select elements by their IDs.

Syntax:

#xyz {
your code...
}

5. > - Child Selector:

Child Selector is used to select the Direct Child of a particular Element.

Syntax:

div > p {
your code...
}

6. <-space-> - Descendant Selector:

Descendant Selector is used to select the Descendants of a particular Element.

Syntax:

h1 p {
your code...
}

7. + - Adjacent Sibling Selector:

Adjacent Sibling selector is used to select the sibling that comes after the certain Element. It is used to select only one Element.

Syntax:

h1 + p {
your code...
}

8. ~ - General Sibling Selector:

General Sibling selector is used to select the general siblings that come after a certain element. It can be used to select one or more than one elements.

Syntax:

nav ~ a {
your code...
}

There are three ways to specify colors:

1. Color Names:

Syntax:

color: red;

2. RGB( red, green, blue):

Syntax:

color: rgb(255,0,0)

3. Hex Code(#RRGGBB):

Syntax:

color: #ff0000

There are two ways to express opacity of an Element:

1. RGBA (red, green, blue, alpha)

color: (255,0, 0, 0.4)

2. Opacity Property:

opacity: 0.8

HSL and HSLA:

HSL and HSLA are also for specifying colors in CSS."h" stands for hue,"s" stands for saturation, and "l" stands for lightness. HSLA also adds another thing "a" that is called alpha, which means opacity.Also note that there is diffrence between lightness and brightness.
